# CSS实现无缝轮播图
逐帧动画配合补间动画实现无缝轮播

逐帧动画利用的是 animation-timing-function 中，的 steps。

```html
<style>
    ul {
        animation: rotate (轮播总时长) steps(轮播个数) infinite;
    }

    @keyframes rotate {
        0% {
            transform: translate(0, 0);
        }
        100% {
            transform: translate(0, calc(-1px * 轮播个数 * 宽度或高度));
        }
    }
    
    li {
        animation: move 单个轮播时长 infinite;
    }
    
    @keyframes move {
        0% {
            transform: translate(0, 0);
        }
        100% {
            transform: translate(0, calc(-1px * 单个宽度或高度));
        }
    }
</style>

<div class="wrapper">
    <ul>
        <li>11</li>
        <li>11</li>
        <li>11</li>
        <li>11</li>
        <li>11</li>
        <li>11</li>
        <!--末尾补一个首条数据-->
        <li></li>
    </ul>
</div>
```
